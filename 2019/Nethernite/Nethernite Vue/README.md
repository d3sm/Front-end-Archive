# Nethernite Vue Test

Test single-page project based on Vue for Nethernite
## Getting started - copy the project

### via https:
 <pre><code>$ git clone https://github.com/d3sm/nether-vue.git</pre></code>

### via ssh:
<pre><code>git@github.com:d3sm/nether-vue.git</pre></code>
<br/> or simply download an archive.

## Install dependencies
<pre><code>npm install</pre></code>

## Check it out!
<pre><code>npm run dev</pre></code>
